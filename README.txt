# Number Guessing Game (Python)

Энэ бол 1–100 хооронд санамсаргүй тоо таах энгийн console тоглоом юм.

## Тоглоомын дүрэм

* Компьютер 1–100 хооронд санамсаргүй тоо сонгоно
* Та **5 удаа** таах боломжтой
* Таасан тоо:

  * Бага байвал → "Их тоо оруулна уу"
  * Их байвал → "Бага бага тоо оруулна уу"
* Зөв таавал → баяр хүргэнэ
* 5 оролдлого дуусвал → зөв хариуг харуулна

## 🛠 Ашигласан технологи

* Python 3
* `random` module

## Яаж ажиллуулах вэ

1. Python суусан эсэхийг шалгах:

```
python --version
```

2. Кодыг ажиллуулах:

```
python main.py
```
